# BossKey

一个基于 WPF 的简洁老板键工具。

## 功能

- 按软件名称保存隐藏目标
- 按窗口标题保存隐藏目标
- 双击来源列表加入保存列表
- 双击保存列表移除目标
- 支持两个全局快捷键
- 老板键：隐藏或恢复目标窗口，同时隐藏或恢复本程序
- 隐藏自身快捷键：仅隐藏或恢复本程序
- 设置自动保存，重启后自动恢复

## 使用方式

1. 启动程序。
2. 在左侧两个来源列表中双击需要的进程或窗口标题，将其加入保存列表。
3. 在右侧设置区分别录入并保存两个快捷键。
4. 按下老板键后，已保存的目标窗口会被隐藏；再次按下会恢复。
5. 按下隐藏自身快捷键后，只会隐藏或恢复本程序自身。

## 开发

环境要求：

- Windows
- .NET 10 SDK

构建命令：

```powershell
dotnet build
```

运行命令：

```powershell
dotnet run
```
